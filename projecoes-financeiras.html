11.2 Recomendações Estratégicas

  1. Prioridade #1 — Migrar para Supabase (R$ 0/mês): A maior vulnerabilidade do sistema é o localStorage. A migração elimina risco LGPD, permite backup e é gratuita no free tier.
  2. Prioridade #2 — App do Paciente (R$ 4,90/paciente): Maior alavanca de receita nova com menor complexidade. Cada 1.000 pacientes ativos = R$ 4.900/mês de receita incremental.
  3. Prioridade #3 — Up-sell para Clínicas: Plano com maior margem (90%). Estratégia: oferecer desconto de 50% no primeiro mês para migração de Evidência para Clínicas.
  4. Prioridade #4 — Marketing de Conteúdo: O ROI de 45× para o usuário é o melhor argumento de venda. Criar calculadora online "Quanto tempo você perde por atendimento?" para gerar leads.
  5. Prioridade #5 — Novos Módulos (Pediatria, CrossFit, Neuro): Ataca o top 1 de contras das personas (22 citações) e abre mercados verticais com baixa concorrência.
